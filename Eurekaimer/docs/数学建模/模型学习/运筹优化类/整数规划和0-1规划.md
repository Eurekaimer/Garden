# 整数规划和0-1规划

## 基本概念

求得的结果或是某个变量只能是整数——也就是整数规划

整数规划也分为纯整数规划和混合整数规划

+ 如果是纯整数规划 比较简单
	+ 特殊的 有些变量只能取0、1也就是所谓的0-1规划
+ 混合整数规划
	+ 这就比较复杂了

整数规划
+ 线性整数规划
	+ Matlab进行求解 在线性规划的基础上，加入决策变量取整数的条件即可
+ 非线性整数规划
	+ 无特定算法 只能使用近似算法 如蒙特卡洛模拟 智能算法
